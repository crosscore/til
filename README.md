# 🏆 GitHub Achievement Lab

GitHubの称号（Achievement）を計画的に収集するための実験リポジトリ。

## 🎯 進捗

| 称号 | 条件 | 状態 |
|------|------|------|
| 🏹 Quickdraw | Issueを5分以内にClose | ⬜ |
| 🧬 YOLO | レビューなしでPRをマージ | ⬜ |
| 👥 Pair Extraordinaire | Co-authoredコミット | ⬜ |
| 🦈 Pull Shark | PRを2件マージ | ⬜ |
| 🦈🦈 Pull Shark ×2 | PRを16件マージ | ⬜ |

## 📋 ルール

- 各ブランチは `feat/`, `fix/`, `chore/` プレフィックスで命名
- PRはmainへマージ
- Co-authoredコミットは `Co-authored-by:` トレーラーを付ける
